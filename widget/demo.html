What is CallNest?

CallNest is a multi-tenant website call conversion platform that enables businesses to instantly turn their website visitors into live calls through an embeddable widget.

ðŸŽ¯ Mission: "Weaponizing websites to convert visitors into calls"

Widget Integration

Each company gets a unique embed code that automatically creates a floating call button on their website:

<script src="https://cdn.callnest.io/widget.js" data-uuid="DEMO-COMPANY-UUID"></script>

The widget automatically:

  • Shows a floating call button (bottom-right by default)
  • Routes calls to company agents
  • Handles agent availability
  • Provides real-time call status

Multi-Tenant Architecture

CallNest supports multiple companies on a single platform:

  • Superadmin: Manages all companies and platform settings
  • Company Admin: Manages their company's agents and widget settings
  • Agent: Receives calls via web panel

Technical Stack

  • Widget: Vanilla JS + WebRTC (future)
  • Frontend: React + Tailwind CSS
  • Backend: Node.js + NestJS
  • Database: PostgreSQL + Redis
  • Real-time: WebSockets
